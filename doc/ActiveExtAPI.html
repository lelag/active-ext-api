Overview

ActiveExtAPI is a module that inject methods to an ActiveRecord model that implements an API that can be used to communicate with an ExtJS Store’s CRUD API. It will add the following 4 methods : 

  - ext_read : used to load a store
  - ext_create : used to create records
  - ext_update : used to update records
  - ext_delete : used to delete records
